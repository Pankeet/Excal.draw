# Excal.com Multi-User Chat

## Overview
This project is a collaborative drawing application inspired by Excalidraw, enhanced with real-time multi-user chat functionality. Users can draw together on a shared canvas while communicating via an integrated chat system.

## Features
- **Real-time Drawing**: Collaborative canvas where multiple users can draw simultaneously.
- **Multi-User Chat**: Integrated chat room for real-time messaging among participants.
- **Responsive Design**: Built with Tailwind CSS for a mobile-friendly interface.
- **Scalable Backend**: Powered by Node.js, Express, Prisma, and PostgreSQL for robust data management.

## Tech Stack
- **Frontend**: Next.js, Tailwind CSS
- **Backend**: Node.js, Express , Websocket
- **Database**: PostgreSQL with Prisma ORM
- **Monorepo Management**: Turborepo

## Installation
1. Clone the repository.
2. Install dependencies: `pnpm install`
3. Set up the database: Configure PostgreSQL and run Prisma migrations.
4. Start the development server: `pnpm run dev`

## Usage
- Access the application in your browser.
- Join a drawing session and start collaborating.
- Use the chat panel to communicate with other users.
